## Install

### Prequisites
1. Python 3.11 or higher
2. Systemd (if you want to run bot as service)
3. Genius client access token

### Basic startup
Clone the repository and install all dependencies by:
```bash
pip install -r requirements.txt
```

```bash
python main.py
```
